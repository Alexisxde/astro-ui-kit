## Component API

### Input

| Prop  | Type   | Description               |
| :---- | :----- | :------------------------ |
| type  | string | El tipo del input.        |
| id    | string | El ID unico.              |
| slot  | string | El nombre del input.      |
| class | string | Clases opcionales de CSS. |
